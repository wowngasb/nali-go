# Nali

给IP加上地理信息的命令行小工具
本项目支持IPv4

## Useage

Query simple IP address:

```bash
nali 2.3.6.7 1.1.2.5
2.3.6.7 <法国 法国>
1.1.2.5 <中国 福建>
```

Query IP addresses from `stdin`:

```bash
ping github.io | nali
```
